# AI Chat Website

## Setup Instructions
1. Clone the repository:
   ```
   git clone https://github.com/atharva2666/ai-chat-website.git
   ```
2. Navigate to the project directory:
   ```
   cd ai-chat-website
   ```
3. Install the dependencies:
   ```
   npm install
   ```
4. Start the development server:
   ```
   npm start
   ```

## Features
- Real-time chat functionality
- User authentication
- Responsive design
- Multi-platform support

## Deployment Guide
1. Build the application:
   ```
   npm run build
   ```
2. Deploy the build folder to your preferred hosting service (e.g., Vercel, Netlify, AWS).

3. Ensure the environment variables are set correctly for production.
